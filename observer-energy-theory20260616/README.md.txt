# 观察者-能量理论（Observer-Energy Theory）

**一个可计算的心理动力学框架。**

用五个状态变量的耦合微分方程描述心理系统的稳态、失稳与修复：

- **情结** = 能量锁住（S锁死，H累积）
- **解离空洞** = 能量切除（化学信号场被抹平至基线）
- **观察者污染** = 安全信号的系统性反向条件化（U<0）
